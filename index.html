Andrew Allen

Front-end Web Developer

Hey, I'm Andrew. I'm a passionate, detail oriented front-end developer who aspires to build responsive pixel perfect websites.

I have a Bachelor's degree in Computer Science from the University of Vermont that gives me a great foundation in building and understanding new technologies.

I love to be creative and see things come to life, which is a huge part in why I fell in love with web development. I feel I'm truly in my element when I'm coming up with solutions to creative problems and constantly learning new things.

When I'm not perfecting my craft, I'm probably making some sounds on my guitar and bass or enjoying the outdoors.

What I Do

Web Development

Using HTML, CSS, Javascript, and various libraries I write semantic, modular front-end code, as well as some back-end code to create mobile-first websites.

Responsive Design

What a time to be alive when you have the world in your pocket. I design and build responsive websites that adapt to any device, browser, or platform.

UX Design

When it comes to a great website, the user comes first. I understand that one too many clicks can make a user leave and that a simple task should be effortless.

UI Design

I believe that a website should look beautiful and modern as well as feel smooth and inviting. I also believe in simplicity and that a website can be "over" designed.
